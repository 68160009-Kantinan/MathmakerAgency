# Assets Directory — The Mathmaker Agency

Place all game image assets in this directory.

## Required Background Images
- bg_menu.png       — Main Menu background
- bg_office.png     — Office Scene background
- bg_winterhall.png — Winterhall Scene background
- bg_snowpath.png   — Snow Path Scene background

## Required Character Sprites
- shin_neutral.png  — Shin (protagonist), neutral expression
- shin_guilt.png    — Shin (protagonist), guilt expression
- rene_happy.png    — Rene (NPC), happy expression
- rene_sad.png      — Rene (NPC), sad expression
- rene_curious.png  — Rene (NPC), curious expression

## Notes
- All images should be PNG format
- Recommended sprite size: 400x600 px (portrait orientation)
- Recommended background size: 1280x720 px (matches game resolution)
- If any asset is missing, FallbackRenderer will draw a colored placeholder automatically
  so the game can run during development without complete assets.
